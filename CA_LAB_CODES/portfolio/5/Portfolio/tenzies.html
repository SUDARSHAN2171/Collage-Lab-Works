Tenzies A Dice Game

You have 10 dice. Goal is have the least amount of rolls possible to get all 10 die faces matching with the same number. This game was made following a React Scrimba course, to practice the fundamentals of state and props when working with functional components.

JavaScript Ecosystem: React framework using Create-react-app template, confetti npm package.

This app is hosted with gh-pages.
