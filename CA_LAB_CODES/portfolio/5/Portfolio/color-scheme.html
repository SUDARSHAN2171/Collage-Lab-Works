The Color Scheme Generator

This color scheme generator was one of my very first projects that I made after learning the basics of asynchronous JavaScript to make GET requests to fetch data. You can choose the color, color pallette and then the color scheme that matches the chosen color will be generated. Once generate is clicked, you will get 5 color options with their respective hex codes. It's possible to copy the hex codes by clicking them.

JavaScript Ecosystem: HTML, CSS, vanilla JS, and a color code API.
