Habit Tracker

This fullstack habit tracker makes it possible to keep track of your daily habits that you want to turn into a routine. For ex: to keep track of how often you exercise or practice a language. With this app, it is possible to add a new habit to your list and keep track of daily habits by registering the day, month, which habit was completed. You can select the month from a drop-down menu to get an overview of which habits you have registered for the current month.

JavaScript Ecosystem: React using Create-react-app Typescript template, Express JS server written in TypeScript, Node.js, and PostgreSQL as the database.
