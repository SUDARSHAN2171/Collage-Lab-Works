Remembr'All Graduation Project

This fullstack app was my group's graduation project for the JavaScript program taken through Brights Learning. Remembr'All is a web application that helps users create custom notifications and reminders based on either a specific time, a location or both. Users can input their notification message, choose a time, date, or location that triggers the notification. The app will keep track of all user-created notifications and create an alert when the conditions set by the user have been met.

JavaScript Ecosystem: React framework using Create-react-app, Google Maps API, Express JS, Node.js, and PostgreSQL.
